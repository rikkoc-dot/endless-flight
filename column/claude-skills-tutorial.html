Claude Skillsの作り方｜10分でAI専門コーチを作る方法

2026年1月19日 公開

Claude Skillsとは、Claudeに専門知識や特定のタスク処理能力を追加できる機能です。マークダウンファイル（SKILL.md）で専門知識を定義することで、普通のChatbotを特定分野の専門家に変えることができます。作成時間は約10分、コーディング不要で対話形式で作成可能です。

Claude Skillsとは何か

Claude Skillsは、Claudeに専門知識を追加するカスタマイズ機能です。従来のChatbotとの違いは、専門知識を体系化して、いつでも呼び出せる形にできる点にあります。

普通のChatbotとの違い

普通のChatbot：

  • 毎回「〇〇について教えて」と質問する必要がある
  • 回答の質がその都度変わる可能性がある
  • 専門的な視点が一貫しない

Claude Skills：

  • 動画をアップロードするだけで自動的に専門家モードに切り替わる
  • 定義した分析項目に沿って一貫した評価が得られる
  • 専門用語や評価基準が統一される

Claude Skillsで作れるもの

Claude Skillsは、あらゆる専門分野のAIコーチを作成できます。

重要なポイント：これらすべてが、コーディング不要で約10分で作成できます。

Claude Skillsの作り方（10分で完成）

1skill-creatorを起動

Claudeに「skill-creatorを使って、〇〇のSkillを作ってください」と依頼します。

例：「skill-creatorを使って、シャドーボクシングのフォーム分析Skillを作ってください」

2要件を対話形式で伝える

Claudeが以下を質問してくるので、順番に答えます：

  • 対象者のレベル（初心者・中級者・上級者）
  • 使用シーン（動画分析・メニュー相談など）
  • 重視するポイント

3SKILL.mdが自動生成される

対話が終わると、Claudeが専門知識を体系化した「SKILL.md」というマークダウンファイルを自動生成します。

内容には、分析ポイント、評価基準、改善アドバイスのテンプレートなどが含まれます。

4パッケージ化してインストール

Claudeが「.skill」ファイルとしてパッケージ化します。このファイルをClaudeにインストールすれば、専門コーチとして使えるようになります。

実際にSkillを使うと何が起きるか

例えば、シャドーボクシングコーチSkillを作成した場合：

使い方

  1. シャドーボクシングの動画をアップロード
  2. Skillが自動起動
  3. スタンス、パンチ、フットワーク、ガード、ヘッドムーブメント、リズムを評価
  4. 10点満点でスコアリング
  5. 優先度順に改善点を提示
  6. 2週間〜3ヶ月のトレーニングプランを提案

重要：これが対話ゼロで自動的に実行されます。毎回「分析してください」と言う必要はありません。

Claude Skillsのメリット

1. 専門知識の体系化

バラバラだった知識を、分析項目・評価基準・改善提案という形で構造化できます。

2. 継続的な改善が可能

Skillは初期バージョンから始まり、使いながら「この項目も追加したい」とフィードバックすることで一緒に成長していきます。

3. 10分で作れる手軽さ

skill-creatorを使えば、対話形式でSkillが完成します。コードを書く必要はありません。

4. 非エンジニアでも作れる

プログラミング知識不要。専門分野の知識さえあれば、誰でもSkillを作成できます。
