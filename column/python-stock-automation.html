Python初心者が株式投資管理を自動化した方法｜プログラミング未経験からの挑戦

2026年1月19日 公開

プログラミング未経験でも、PythonとAI（Claude）を活用すれば株式投資管理を自動化できます。銘柄の株価、配当利回り、ネットキャッシュ比率などを自動取得し、Googleスプレッドシートでの手作業を完全に不要にすることが可能です。

プログラミング未経験が直面した課題

株式投資をしていると、以下のような作業を毎日繰り返していました：

  • 各銘柄の現在株価をWebサイトで確認
  • Googleスプレッドシートに手入力
  • 過去5年高値からの凋落率を計算
  • 配当利回り、自己資本比率を転記
  • ネットキャッシュ比率の高い銘柄をスクリーニング

この作業、毎日30分〜1時間かかっていました。

Pythonでできることに気づいた瞬間

筑波大学のPython教材を眺めていたとき、「Webスクレイピング」という章に目が止まりました。

Webサイト内を巡回して、情報を自動で集める――これができるなら、株価データも自動取得できるのでは？

そこで、AI（Claude）に質問してみました：

「Pythonを使って、銘柄コード一覧から株価・配当利回り・ネットキャッシュ比率を自動取得し、CSVで出力できますか？」

返ってきた答えは、あっさり。「できます」

Before / After：何が変わったか

❌ Before（手作業）

  • 毎日30分〜1時間の作業
  • 転記ミスのリスク
  • スクリーニング条件の変更が面倒
  • 過去データとの比較が困難

✅ After（Python自動化）

  • ボタン1つで完了（数秒）
  • 転記ミスゼロ
  • スクリーニング条件を即座に変更可能
  • 過去データとの比較が簡単

実装できた機能

✅ 自動取得データ

  • 現在株価
  • 過去5年高値
  • 凋落率（過去5年高値からの下落率）
  • 配当利回り
  • 自己資本比率
  • ネットキャッシュ比率

✅ 出力形式

  • CSV形式（Excel / Google Sheets対応）
  • 銘柄コード一覧をCSVで用意するだけ
  • Webアプリ化も可能（Claudeが提案）

プログラミング未経験でもできた理由

1. AIに聞きながら進めた

わからないエラーはClaudeに貼り付けて質問。「このエラーの意味は？」「どう修正すればいい？」と聞けば、すぐに解決方法を教えてもらえました。

2. 完璧を目指さなかった

Pythonの文法やロジックを完全に理解しようとせず、「動けばOK」という割り切りで進めました。理解は使いながら深めていけばいいと考えました。

3. 具体的な目的があった

「Pythonを勉強する」ではなく「株式投資管理を楽にする」という明確な目的があったため、モチベーションが続きました。

Pythonで自動化する3つのメリット

メリット1: 時間の節約

毎日30分〜1時間かかっていた作業が数秒で完了。年間で180時間以上の節約になります。

メリット2: データの正確性

手入力による転記ミスがゼロに。常に最新の正確なデータで判断できます。

メリット3: スクリーニング条件の柔軟性

「ネットキャッシュ比率50%以上」「配当利回り3%以上」など、条件を即座に変更してスクリーニングできます。

最初につまずいたポイント

1. インストールでつまずく

教材がWindows前提だったのに対し、自分の環境はMac。Pythonのインストール、ターミナル、パスの設定で最初からつまずきました。

解決方法：「Mac Python インストール」でGoogle検索し、AIに聞きながら進めました。

2. 教材の内容がほとんど理解できない

筑波大学の教材を読んでも、専門用語だらけで理解できませんでした。

解決方法：コード部分だけコピーして実行。「動いた」「出力された」という体験を先に積みました。

3. スクレイピングの仕組みがわからない

Webサイトから情報を取得する仕組みが理解できませんでした。

解決方法：Claudeに「株価を自動取得するコード例を教えて」と質問。すぐに動くコードを提供してもらえました。
